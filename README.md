# Tilemap
Tilemap on the GPU (C++/OpenGL)

> Concept by [Tojicode](https://blog.tojicode.com/2012/07/sprite-tile-maps-on-gpu.html).

# Dependencies
## GNU/Linux X11 tested on Ubuntu 18.04
* libx11-dev
* libxrandr-dev
* libxinerama-dev
* libxcursor-dev
* libxi-dev
* libgl1-mesa-dev

# Build

```console
git clone --recursive https://github.com/Loulfy/tilemap
cd tilemap
mkdir build
cd build
cmake ..
make
```
